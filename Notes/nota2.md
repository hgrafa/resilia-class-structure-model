# Funções

## Conhecimentos prévios

- variáveis
- array, vetores, matrizes
- repetição e seleção

## Lembranças

- função 1º grau
- f(x) , f(5)
- retorno

```
  f(x) = x + 3
  f(x, y) = x + y;
  f(2) = 2 + 3
  f(2) = 50
  mmc(12, 9, 3)
```

```java
double f(double x){
  return x + 3;
}

double somar(double a, double b){
  return a + b;
}
```

função é um **processo** que encapsula outras ações.
